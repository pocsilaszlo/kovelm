# Követelményelemzés Kvízalkalmazás

Egy interaktív kvízalkalmazás a követelményelemzés témakörében.

## Funkciók

- 122 kérdés véletlenszerű sorrendben
- Válaszlehetőségek véletlenszerű sorrendben
- Több válaszos és egy válaszos kérdések támogatása
- Valós idejű pontszámolás
- Végösszefoglaló helytelen válaszokkal
- Újraindítás funkció

## Telepítés

```bash
npm install
```

## Fejlesztés

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Netlify Deploy

Az alkalmazás Netlify-re deployolható. A `netlify.toml` fájl tartalmazza a szükséges konfigurációt.

1. Csatlakoztasd a GitHub repository-t a Netlify-hez
2. A build command: `npm run build`
3. A publish directory: `dist`
